ROE DuPont Decomposition from Historical SEC Filings

This project provides a Python tool that automatically extracts financial data from SEC 10-K filings and performs ROE decomposition using the DuPont framework.

Quick start

pip install -r requirements.txt
python ROE_DuPont_Decomposition.py

Project files

ROE_DuPont_Decomposition.py   Main analysis script
Find_Nonstandard_Labels.py   Label inspection tool
Labels_Library.py            Label library
